MembersController
controller in module app

Description

The MembersController is a controller that is used to expose functionality required for members page. It provides methods to manage UI as well as watching for changes in member details and persisting changes using meetupService. Also MembersController is responsible for updating selected user details according to active route parameters.

Dependencies

  • $scope
  • $routeParams
  • $mdSidenav
  • meetupService
