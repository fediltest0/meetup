refService
service in module app

Description

The refService is used to initialise and cache Firebase reference objects.

This is a low-level data access service that in most cases should not be used directly in controllers or directives. refService should be used by high-level services (like app.meetupService) that provide abstraction on data source and add application specific semantics.

For more information about Firebase reference objects see Firebase API reference

Dependencies

  • $window
  • baseFirebaseUrl

Methods
